General Chord Charts

Note: "General Chord Charts" refers to charts that have notable chord patterns, but do have an emphasis on any specific patterns (such as chord-rolls or chord-trills).
For charts that target more specific chord skill-sets, scroll down.

Level 16

Waxing and Wanding [EXH]



Stats

BPM: 191

Notes per second: 6.33

Links: sdvx.in link // Remywiki link // Chart video (Player: DDX)


Description

Waxing and Wanding EXH contains chords for a large portion of its runtime, most notably in measures 83-86.
Since it's a 16, the chords are slow, which makes it a good starting point for players trying to learn chords for the first time.


Level 17.0

Electronic Sports Complex [EXH]



Stats

BPM: 220

Notes per second: 7.33

Links: sdvx.in link // Remywiki link // Chart video (Player: DDX)


Description

Esports Complex is a relatively easy 17, with a notably long chord section at the end.
While the chord pattern itself is complex compared to other charts of this level, it's rather slow.
Overall, it's a good practice chart for players who are trying to learn chord patterns that utilize FX.


Level 17.5

Booths of Fighters [EXH]



Stats

BPM: 155-225 (225 during the chords)

Notes per second: 11.2

Links: sdvx.in link // Remywiki link // Chart video (Player: DDX)


Description

By far the most well-known chord chart in the game, BoF EXH has appeared as a 10th-dan boss song for every game since GRAVITY WARS, containing rather awkward one-hand patterns and a massive chord section.
The infamous 17-measure long chord section is filled with 12ths at 225 BPM, which is extremely fast compared to other 17s.
Despite how fast it is, the actual pattern is straightforward once you learn it.

Waxing and Wanding [MXM 18.1]



Stats

BPM: 191

Notes per second: 6.33

Links: sdvx.in link // Remywiki link // Chart video (Player: DDX)


Description

Like Comet Popcorn, Waxing MXM is another relatively straightforward 18 that spikes in difficulty near the ending.
Waxing and Wanding's most notable pattern' a very long chord section from measures 59-71 and 83-90 that includes lots of FX+BT chords and 3-note chords.
While the chords aren't fast, its chord section is *very* long, and the chord patterns are very complex.
It might be difficult to read on sightread, but is a very good practice chart for complex chord patterns.


Star☆Beat [MXM 18.1]



Stats

BPM: 185

Notes per second: 9.25

Links: sdvx.in link // Remywiki link // Chart video (Player: DDX)


Description

Star Beat is another low-end 18 that spikes in difficulty with a chord section.
Be warned that while the long periods of non-moving lazers during the chords will inflate your score, they also make it harder to read the chords when they sit right on top of the lanes. Make sure you're good at reading chords before practicing on this chart.


Tickled Pink [MXM 18.3]



Stats

BPM: 180

Notes per second: 9.0

Links: sdvx.in link // Remywiki link // Chart video (Player: DDX)


Description

While Tickled Pink is a mid-range 18 that mostly focuses on one-handing and lazers, it does contain a sudden burst of chords at measures 72 and 74.
Be prepared for it, as it might catch you off-guard on sightread!


Xepher Light and Darkness Dragon REMIX [GRV 18.4]



Stats

BPM: 180

Notes per second: 12.0

Links: sdvx.in link // Remywiki link // Chart video (Player: DDX)


Description

Xepher REMIX is a rather intense 18, as it's filled with streams and awkward one-handing.
It notably contains an 8-measure long section of 16th-note chord patterns, with heavy usage of alternating between FX and BT.
This chart is a great way to thoroughly test your ability to read complex chords.


Warriors Aboot [EXH 18.4]



Stats

BPM: 175

Notes per second: 11.66

Links: sdvx.in link // Remywiki link // Chart video (Player: DDX)


Description

Warriors Aboot EXH is a mid-to-high-range 18 that contains a large 16th-note chord section from measures 53-73.
Be warned that it additionally contains some rather tricky patterns and some jacks during the beginning half of the chart.


Level 18.5-18.9

For UltraPlayers [EXH 18.6]



Stats

BPM: 201

Notes per second: 13.4

Links: sdvx.in link // Remywiki link // Chart video (Player: DDX)


Description

UltraPlayers EXH is a difficult 18 which contains (among other things) a very difficult burst of chords that are arranged in a rather strange pattern, which may make it hard to read on sightread.
Besides the chords, this chart is a general all-rounder in terms of difficult patterning: it targets a wide variety of skill-sets, so be prepared.
